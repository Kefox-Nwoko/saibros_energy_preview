# Saibros Energy - Vessel Management System

## 🚢 100% FREE Vessel Management System

A complete vessel management solution with admin panel, API, and website integration.

**No subscriptions. No time limits. No database required.**

---

## 🚀 Quick Start

1. **Install XAMPP** (if not installed)
   - Download: https://www.apachefriends.org/

2. **Copy files to htdocs**
   ```
   C:\xampp\htdocs\saibros\
   ```

3. **Start Apache in XAMPP**

4. **Access Website**
   ```
   http://localhost/saibros/index.html
   ```

5. **Access Admin Panel**
   - **Easy way:** Click the "Admin" button (⚙️) in the navbar
   - **Direct URL:** `http://localhost/saibros/admin/`
   - **Login:** admin / saibros2024

## ✨ Features

### Admin Access
- ✅ **Admin button in navbar** - Click gear icon (⚙️) on any page
- ✅ Easy access to vessel management backend
- ✅ Add/Edit/Delete vessels with images

### Translation
- ✅ **Working language switcher** - Globe icon (🌐) in navbar
- ✅ English, French, Portuguese support
- ✅ Google Translate integration

### Vessel Management
- ✅ Admin panel for vessel management
- ✅ Image upload functionality
- ✅ Status management (Available, On Charter, Maintenance)
- ✅ REST API endpoint
- ✅ JSON-based storage (no database needed)
- ✅ Responsive design
- ✅ 4 vessels pre-loaded
